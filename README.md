# Full Stack Login Application  
### React Frontend + Spring Boot Backend

This is a simple full stack login & signup application built using:

- **React (Frontend)**
- **Spring Boot (Backend)**
- **MySQL Database**
- **REST API communication**


## Features

### ✔ Frontend (React)
- Login form  
- Signup form  
- API communication using `fetch()`  
- Beautiful UI  

### ✔ Backend (Spring Boot)
- User signup  
- User login  
- Prevent duplicate email registration  
- Connected to database  
- CORS enabled for frontend
